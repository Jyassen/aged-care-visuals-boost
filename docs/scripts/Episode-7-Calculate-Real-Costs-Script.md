**Q: "Where do I find a plan's formulary?"**

**Q: "Should I include dental and vision in this?"**
A: "Good question! Original Medicare doesn't cover routine dental/vision, so budget separately for those ($500-2,000/year typically). Some Advantage plans include it - factor that in if comparing!"

**Q: "What if I have a really expensive year with surgery?"**
